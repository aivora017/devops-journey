# week 1: Linux Basics
## Commands
### pwd
Shows current directory.


### ls
Lists files and folders.

### cd
Change directory.

## Permissions

- chmod 644 = rw-r--r--
- chmod 755 = rwxr-xr-x
- chmod 700 = rwx------


## Useful Tools

- grep = search text
- pipes | = connect commands

## Script Execution
```bash
chmod +x file.sh
./file.sh
```
